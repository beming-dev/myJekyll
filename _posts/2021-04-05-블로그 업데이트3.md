# 비 내리는 효과 추가

header부분에 비내리는 효과를 추가했다. 사실 별로 안예쁜것 같긴한데 그냥 한번 이런거 해보고 싶었다. 내 블로그니까! 

그래도 pc에서 보면 나름 괜찮은데 스마트폰에서 보면 비도 굵고, 높이도 안 맞고, 떨어지는 속도도 이상한 것 같고 비 개수도 이상하다. 왜 그런지는 도저히 모르겠다. 그냥 스마트폰에서는 안보이도록 해놓아야겠다.

비 내리는 효과는 canvas와 javascript로 만들었는데, 전에 interactive coding에 빠졌을 때 만들었던 비 관련 프로젝트를 살짝 수정해서 집어넣었다.

[github](https://github.com/beming-dev/Interactive-develop/tree/main/rainFalling)

canvas를 추가하는 과정에서  canvas가 뒤의 요소를 가려서 클릭이 안되는 문제가 발생했는데, 이를 해결하다가 z-index라는 css속성을 알게 됐다. z-index값이 작을수록 요소가 뒤에 배치되는데, 되게 유용한 것 같다.



# SEO

전에 검색 엔진 최적화를 해둬서 beming-dev를 구글에 검색하면 검색이 됐었다. 근데 내 블로그 글들은 아무리 검색해도 안나오길래 이상하다 했는데, 이번에 그 이유를 찾았다.

[google search-console](https://search.google.com/search-console/about) 에서 색인 생성 범위를 보는데, 오류 난 것들이 보였다.

<img src="/assets/images/화면 캡처 2021-04-04 224334.png"/>

그래서 클릭해서 보니 url에 .html이 2개씩 들어가있었다.

<img src="/assets/images/화면 캡처 2021-04-05 130250.png"/>

그래서 sitemap.xml 파일에 들어가서  사이트의 url부분을 바꿔줬다.

```
<loc>{{ site.url }}{{ post.url }}.html</loc>
```

이렇게 돼있던걸

```
<loc>{{ site.url }}{{ post.url }}</loc>
```

이렇게 바꿔줬더니 이제 키워드를 통해서도 내 블로그가 검색된다.

<img src="/assets/images/화면 캡처 2021-04-05 130250.png"/>

내 블로그가 검색되니까 기분이 좋기도 혹시 글에 잘못된 정보가 있으면 어쩌지란 생각도 좀 들었다. 앞으로 글을 쓸 때 좀 더 신중히 써야겠다.
