## css

css도 딱히 특별한 점은 없었지만, 3가지 정도 생각해 볼만한게 있었다.

1. main페이지에 들어갈 때, animation속성을 사용해 header가 꽉 찼다가 줄어드는 효과를 주었다. 사이트에 처음 들어갔을때의 index.html과 연속적인 느낌을 주기 위해서 일부러 그렇게 했다.

2. 그라데이션을 사용해봤다.

    ```css
    background: linear-gradient(to right, #ffdde1, #ee9ca7)
    ```

    위와같이 background나 background-color에 속성을 줌으로써 사용할 수 있다. 그라데이션을 직접 사용해본 것은 이번이 처음이었다. 그라데이션은 잘못 사용하면 엄청 이상하기 때문에 그라데이션을 추천해주는 사이트를 참고했다.

    [그라데이션](https://uigradients.com/)

3. 마우스를 따라다니는 공을 만들기 위해 canvas로 화면 전체를 덮었는데, canvas가 화면을 다 가려서 페이지의 링크들이 눌리지 않았다. 

   처음에는 z-index: -1을 사용해 canvas를 뒤로 빼려했는데, canvas가 뒤로 빠지니 마우스를 따라다니는 녀석이 이미지 나 header같은 곳을 지날때 가려져서 보이지 않았다. 

   그래서 구글링 해본 결과, 

   ```css
   pointer-events: none;
   ```

   이라는 새로운 속성을 알게 되었다. 이 속성을 적용하면 그 element는 마우스의 입력 대상이 되지 않아서 뒤의 요소를 클릭할 수 있게 된다.



## javascript

js로는 마우스를 따라다니는 원을 만들었다.

jquery를 사용해보진 않아서 잘 모르지만, jquery를 사용하면

```javascript
$(document).mousemove(function(e) {
      mouseX = e.pageX;
      mouseY = e.pageY;
      $('#aa').css("left", mouseX+15).css("top", mouseY+15).html("x:"+mouseX+" y:"+mouseY);
});
```

이런식으로 간단하게 마우스를 따라다니는 효과를 줄 수 있는 것 같다.

하지만 마우스를 따라다니는 속도도 조절하고 싶고, jquery파일을 프로젝트에 추가하기도 싫어서 그냥 js로 간단하게 만들었다.

아래는 내가 짠 코드다.

```javascript
let canvas = document.createElement('canvas');
document.body.appendChild(canvas);
let ctx = canvas.getContext('2d');

let pixelRatio = window.devicePixelRatio > 1 ? 2 : 1;

let width = document.body.clientWidth;
let height = document.body.clientHeight;

window.addEventListener('resize', resize);

let mouseX;
let mouseY;
window.addEventListener('mousemove', onMove);

let realX = 0;
let realY = 0;

resize();

window.requestAnimationFrame(animate);

function resize(){
    width = window.innerWidth;
    height = window.innerHeight;

    canvas.width = width * pixelRatio;
    canvas.height = height * pixelRatio;

    ctx.scale(pixelRatio, pixelRatio);
}

function animate(){
    window.requestAnimationFrame(animate);

    realX += (mouseX - realX) * 0.12 || 0;
    realY += (mouseY - realY) * 0.12 || 0;

    ctx.clearRect(0, 0, width, height);

    ctx.fillStyle = '#FBCCD1';
    ctx.beginPath();
    ctx.arc(realX, realY, 7, 0, Math.PI*2);
    ctx.fill();
}

function onMove(e){
    mouseX = e.clientX;
    mouseY = e.clientY;
}
```

마우스의 위치를 mousemove이벤트로 계속 받아오고, 현재 공의 위치를 기억해 뒀다가 현재 공의 위치에서 마우스의 위치와 공의 위치의 차이만큼의 값에 특정값을 곱한 후 계속 더하는 방식이다. 

나는 0.12를 곱해서 사용했더니 맘에 드는 속도가 나왔다.
